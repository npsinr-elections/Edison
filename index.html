Edison

An implementation of an EVM in HTML, CSS, JS, and Python.

View project on GitHub

The new face of elections

Edison is the perfect app to hold the perfect elections for an audience of people. The user interface is designed to be appealing, while increasingly user-friendly, ensuring the viewer, and the user an incredible, unique experience.

Customization at its best

Create offices, add candidates to offices, add candidate images, customize background colors and font colors: You name it, Edison can do it.

Compatible on all platforms with a browser!

Edison can work on any machine with Python 2.7 (or above) and the Bottle Python library installed. A modern browser is required to run the app. A portable executable is available for windows which require NO pre-installation, and can be run directly after download.
